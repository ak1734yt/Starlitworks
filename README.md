# Starlit Siege Works 🌟

A premium, high-performance freelancing platform designed for advanced service management, secure asset delivery, and real-time client engagement.

## 🚀 Key Features

- **AI Fraud Detection**: Rule-based heuristic engine to detect VPNs, proxies, and suspicious access patterns.
* **The Starlit Vault**: Secure, client-specific digital asset delivery (Bot tokens, license keys, etc.).
- **Live System Health**: Real-time monitoring of infrastructure and security status.
- **Interactive Roadmap**: Dynamic project tracking from Briefing to Live deployment.
- **Global Pulse Intelligence**: Comprehensive administrative oversight with geolocation verification.
- **Smart Invoicing**: Automated monthly installment tracking and financial radar.
- **2FA Security**: Two-factor authentication for administrators and managers.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Framer Motion, Tailwind CSS, Lucide React.
- **Backend**: Node.js, Express, SQLite (Better-SQLite3).
- **Authentication**: Passport.js (Google, Discord, Microsoft, Apple), JWT.
- **Notifications**: Real-time polling engine + Discord Webhooks (Vaibhav Relay).

## 📦 Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   cd backend && npm install
   ```
3. Set up your `.env` file in the `backend/` directory.
4. Run the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

This project is optimized for deployment on **Vercel**. 
- Link your GitHub repository.
- Configure your Environment Variables in the Vercel dashboard.
- Ensure your database (SQLite) is migrated to a persistent provider like Supabase or Neon for production use.
